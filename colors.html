Less colors by Jason Robb. Making visual design in the browser a bit easier

Creating background colors or slightly faded text is pretty easy with Less. But it could be easier.

Fork it at Github, or something like that.

// Colors.less
// A variety of colors and transparency
// to make designing in the browser a lot easier
// ---------------------------------------------

@white: #fff;
@black: #000;

// White
@white1:  fade(@white, 1%);
@white2:  fade(@white, 2%);
@white3:  fade(@white, 3%);
...
@white97: fade(@white, 97%);
@white98: fade(@white, 98%);
@white99: fade(@white, 99%);

// Black
@black1:  fade(@black, 1%);
@black2:  fade(@black, 2%);
@black3:  fade(@black, 3%);
...
@black97: fade(@black, 97%);
@black98: fade(@black, 98%);
@black99: fade(@black, 99%);


// Mix for gray scale
@gray99: mix(@black, @white, 1%);
@gray98: mix(@black, @white, 2%);
@gray97: mix(@black, @white, 3%);
...
@gray3: mix(@black, @white, 97%);
@gray2: mix(@black, @white, 98%);
@gray1: mix(@black, @white, 99%);
