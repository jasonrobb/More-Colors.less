White, black, and grayscale variables for designing in the browser.

White, Black, and Grayscale

White to transparent

// @white##
@white1:  fade(@white, 1%);
...
@white99: fade(@white, 99%);

Mix white with the background color. The further from @white99 you go, the more transparent the White. I've excluded @white100 and @white0 because the former would be solid white and the latter would be completely transparent.

Black to transparent

// @black##
@black1:  fade(@black, 1%);
...
@black99:  fade(@black, 99%);

Mix black with the background color. The further from @black99 you go, the more transparent the Black.

Solid grayscale

// @gray##
@white1:  fade(@white, 1%);
...
@white99: fade(@white, 99%);

Solid grayscale colors. @gray99 is closest to white, while @gray1 is closest to black.

Best used in...

Try using these variables to:

  • set the background-color a little lighter or darker
  • making <hr /> with a light top border and a dark bottom border
  • use for mixing colors like this: mix(@color, @white50, 50) which provides an additional point of adjustment. Hey, it's helped me.

White to transparent

Input

// White
@white: #fff;

@white1:  fade(@white, 1%);
@white2:  fade(@white, 2%);
@white3:  fade(@white, 3%);
...
@white97: fade(@white, 97%);
@white98: fade(@white, 98%);
@white99: fade(@white, 99%);

Output

// White
@white: #fff;

@white1: rgba(255, 255, 255, 0.01);
@white2: rgba(255, 255, 255, 0.02);
@white3: rgba(255, 255, 255, 0.03);
...
@white97: rgba(255, 255, 255, 0.97);
@white98: rgba(255, 255, 255, 0.98);
@white99: rgba(255, 255, 255, 0.99);

Black to transparent

Input

// Black
@black: #000;

@black1:  fade(@black, 1%);
@black2:  fade(@black, 2%);
@black3:  fade(@black, 3%);
...
@black97: fade(@black, 97%);
@black98: fade(@black, 98%);
@black99: fade(@black, 99%);

Output

// Black
@black: #000;

@black1: rgba(0, 0, 0, 0.01);
@black2: rgba(0, 0, 0, 0.02);
@black3: rgba(0, 0, 0, 0.03);
...
@black97: rgba(0, 0, 0, 0.97);
@black98: rgba(0, 0, 0, 0.98);
@black99: rgba(0, 0, 0, 0.99);

Shades of gray

Input

// Solid grayscale
@gray99: mix(@black, @white, 1%);
@gray98: mix(@black, @white, 2%);
@gray97: mix(@black, @white, 3%);
...
@gray3: mix(@black, @white, 97%);
@gray2: mix(@black, @white, 98%);
@gray1: mix(@black, @white, 99%);

Output

// Solid grayscale
@gray1: #fcfcfc;
@gray2: #fafafa;
@gray3: #f7f7f7;
...
@gray97: #080808;
@gray98: #050505;
@gray99: #030303;

Why do this?

Thanks to LessCSS I can do my color exploration in the browser, instead of the Photoshop color palette. Sure, I could be using hsla(0,0,0,0.5), but that's a lot more characters and shift key usage than just typing in @black50 and being done with it.

Aren't there better ways to do this?

Probably.

Why not just use lighten(@color, 20);

Because that's quite a few more characters than typing @white20.
