White, black, and grayscale variables for easier color manipulation while you design in the browser.

White, Black to transparent

Use any variable from @white1 to @white99, the lower the number, the more transparent the color.

Black and white work the same way. The higher the suffix number, the more solid (i.e. not transparent) the color will be.

Color: #09c
@white10 @white30 @white50 @white70 @white90
@black10 @black30 @black50 @black70 @black90 
// syntax: 	@white##
// 		@black##

@white: #fff;

@white1:  fade(@white, 1%);
@white2:  fade(@white, 2%);
@white3:  fade(@white, 3%);
@white4:  fade(@white, 4%);
...
@white96: fade(@white, 96%);
@white97: fade(@white, 97%);
@white98: fade(@white, 98%);
@white99: fade(@white, 99%);

Solid grayscale colors

Grayscale colors are solid. @gray99 is closest to white, while @gray1 is closest to black.

Color: #808080 (@gray50)
@gray10 @gray20 @gray30 @gray40 @gray50
@gray60 @gray70 @gray80 @gray90 @gray97 
// syntax: @gray##
@gray99: mix(@black, @white, 1%);
@gray98: mix(@black, @white, 2%);
@gray97: mix(@black, @white, 3%);
@gray96: mix(@black, @white, 4%);
...
@gray4: mix(@black, @white, 96%);
@gray3: mix(@black, @white, 97%);
@gray2: mix(@black, @white, 98%);
@gray1: mix(@black, @white, 99%);

Best used in...

Try using these variables to:

  • set the background-color a little lighter or darker
  • making <hr /> with a light top border and a dark bottom border
  • use for mixing colors like this: mix(@color, @white50, 50) which provides an additional point of adjustment. Hey, it's helped me.

Why do this?

Thanks to LessCSS I can do my color exploration in the browser, instead of the Photoshop color palette. Sure, I could be using hsla(0,0,0,0.5), but that's a lot more characters and shift key usage than just typing in @black50 and being done with it.

Are there better ways to do this? Probably. Don't care.
